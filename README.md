# pageWatch
Simple python script that monitors a webpage for changes.

## Usage

`git clone https://github.com/heyseth/pageWatch.git` or download and extract the zip.

`cd pageWatch`

`pip install BeautifulSoup4`

`python pageWatch.py`

Exit at any time with control-c.
